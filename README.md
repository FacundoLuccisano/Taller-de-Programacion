# Taller de Programación

Repositorio con ejercicios, parciales y trabajos prácticos
realizados durante la cursada de la materia **Taller de Programación**
de la Licenciatura en Sistemas – Universidad Nacional de La Plata.

El objetivo de este repositorio es documentar el proceso de aprendizaje
en distintos paradigmas de programación, a nivel académico.

## Contenidos

### Programación Imperativa
- Lenguaje: Pascal
- Estructuras básicas
- Uso de listas y registros
- Resolución de parciales y ejercicios prácticos

### Programación Orientada a Objetos
- Lenguaje: Java
- Clases, objetos y métodos
- Encapsulamiento y modularización
- Ejercicios y parciales académicos

### Programación Concurrente
- Lenguaje: R-Info
- Comunicación entre procesos
- Sincronización y coordinación
- Resolución de ejercicios prácticos

## Estado
Repositorio en constante actualización a medida que avanzo en la carrera.
